# LOGBOOK

Development session logs following the [ADC (Autonomous Distributed Coordination)](https://github.com/zembutsu/voice-harmony-protocol) approach.

## Purpose

This directory contains session-based development logs that capture:
- Decision points and rationale
- Technical discoveries and insights
- Voice interactions (Human ↔ AI collaboration)
- Handoff context for session continuity

## Recording Style

This project uses **Factorio-livelog style** with hierarchical IDs, decision points (★), and distance tracking.

## File Naming Convention

```
S{NNN}_en.md  - English version (primary)
S{NNN}_ja.md  - Japanese version (secondary)
```
